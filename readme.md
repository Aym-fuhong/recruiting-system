## 使用说明：

### clone

```
git clone git@github.com:linwenjun/recruiting-system.git
```

### 配置

切换gh-pages分支

```
git checkout gh-pages
```

安装npm依赖包

```
npm install
```

第一次使用手动build项目

```
gulp build
```

### 使用

启动server

```
gulp server
```

用浏览器启动

```
http://localhost:8080/
```

### 注意

** public目录里的文件通过source自动生成，请勿手动修改。 **
